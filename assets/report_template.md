# Структурный трафарет идеального отчёта ИТК

Это визуальный шаблон-заготовка готового отчёта по учебной практике.
Скопируй его и заменяй текст в `[[…]]`-плейсхолдерах своим содержимым.
**Не путать с DOCX-шаблоном ученика** — этот файл показывает только
порядок и тип контента; DOCX-шаблон даёт правильные рамки и штампы.

<placeholder_legend>
- `[[PLAIN: …]]`           — обычный текст, подставить как есть.
- `[[PARA: …]]`            — целый абзац основного текста (TNR 14,
                              justify, отступ 1.25, межстрочный 1.0).
- `[[FIGURE: путь | подпись]]` — рисунок + подпись «Рисунок-N.M …».
- `[[TABLE: подпись | строки]]` — таблица + подпись «Таблица-N.M …».
- `[[REF: N]]`             — ссылка на источник `[N]`.
- `[[PAGE]]`               — фактический номер страницы для СОДЕРЖАНИЯ.
</placeholder_legend>

## ТИТУЛЬНЫЙ ЛИСТ

> Берётся из шаблона ученика без изменений структуры. Меняй только
> текстовые поля: тема, ФИО студента, курс, группа, специальность,
> квалификация, период практики, ФИО руководителя.

```
ИННОВАЦИОННЫЙ ТЕХНИЧЕСКИЙ КОЛЛЕДЖ ГОРОДА АЛМАТЫ

ОТЧЕТ
по учебной практике

«[[PLAIN: ТЕМА ОТЧЁТА]]»

Студент:        [[PLAIN: ФАМИЛИЯ ИМЯ ОТЧЕСТВО]]
Курс:           [[PLAIN: N]]
Группа:         [[PLAIN: РПОN-NNР]]
Специальность:  [[PLAIN: КОД И НАЗВАНИЕ]]
Квалификация:   [[PLAIN: …]]
Период практики: с «[[PLAIN: ДД]]» [[PLAIN: месяц]] [[PLAIN: ГГГГ]]
                по «[[PLAIN: ДД]]» [[PLAIN: месяц]] [[PLAIN: ГГГГ]]

Руководитель:   [[PLAIN: ФИО]]
Оценка: ____________  Дата защиты: «___» __________ 20__ г.

Алматы, [[PLAIN: ГГГГ]]
```

> ❗ Рамки и штампа на этой странице **нет**.

## Памятка по объёму

<volume_check>
- Стандартная страница ≈ 28–32 строки основного текста.
- Один подраздел с одним рисунком ≈ 1.5–2 страницы.
- Минимальный объём отчёта: **27 страниц** (без приложений).
- Чтобы попасть в объём: 5 подразделов в 1 ОСНОВНОЙ ЧАСТИ + 4 в
  2 СПЕЦИАЛЬНОЙ + введение + заключение + список ≈ 30–35 страниц.
</volume_check>
