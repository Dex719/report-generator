# report-generator

Claude Code skill для генерации учебных DOCX-отчётов с правильными ГОСТ-рамками и штампами.

## Что делает

- Берёт DOCX-шаблон ученика (с VML/DrawingML рамками и штампами)
- Подменяет текстовые поля (ФИО, тема, группа, даты)
- Пересобирает тело документа: СОДЕРЖАНИЕ → ВВЕДЕНИЕ → ОСНОВНАЯ ЧАСТЬ → СПЕЦИАЛЬНАЯ ЧАСТЬ → ЗАКЛЮЧЕНИЕ → СПИСОК ЛИТЕРАТУРЫ
- Конвертирует в PDF через LibreOffice
- Проверяет по чек-листу перед выдачей

## Структура репозитория

```
SKILL.md                      — инструкция для Claude Code (навык /report-generator)
assets/
  build_from_student.py       — скрипт-скелет Python для сборки DOCX
  report_template.md          — структурный трафарет отчёта с плейсхолдерами
references/
  report_rules.md             — полные правила генерации (окружение, форматирование, антипаттерны)
```

## Использование

1. Установи [Claude Code](https://claude.ai/code)
2. Открой папку репозитория в Claude Code
3. Вызови навык: `/report-generator`
4. Подготовь принятый отчёт-эталон (`.docx`) — без него генерация невозможна

## Требования к окружению

```bash
python3 -m venv ~/report_env
~/report_env/bin/python -m pip install python-docx lxml Pillow matplotlib
sudo apt-get install -y libreoffice poppler-utils
```

## Ключевой принцип

Рамки и штампы **не создаются программно** — они копируются из шаблона ученика байт-в-байт. Любая попытка нарисовать их через `python-docx` Table API даёт визуально неточный результат.

## Формат отчёта

- Шрифт: Times New Roman 14 (подписи — 12)
- Поля: 30/10/15/30 мм
- Межстрочный: 1.0, абзацный отступ: 1.25 см
- Объём: ≥ 27 страниц
- Литература: ≥ 15 источников
